# E-Commerce Portal

This project is a simple E-Commerce web application similar to Flipkart or Amazon.  
It is developed as part of Assignment 3.

## 🔹 Features

1. Navigation menu for entire website:
   - Home
   - Products
   - Contact Us
   - About Us

2. User Authentication:
   - User Registration form
   - Login form
   - Input field validation

3. User Services:
   - Orders list
   - Wishlist
   - Comments & Reviews
   - Customer Care information
   - Reporting options

## 🔹 Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- JSON (for database)

## 🔹 How to Run the Project

1. Install Node.js
2. Open project folder in terminal
3. Run:

   npm install
   npm start

4. Open browser and go to:

   http://localhost:3000
